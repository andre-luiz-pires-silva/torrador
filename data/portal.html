%

O Artisan comanda a potência de 0 a 100. Nesta fase o queimador é liga/desliga: ele acende quando a potência pedida ultrapassa este valor (ex.: 50 acende acima de 50%).

Precisa de ajuda para conectar no Artisan?

A configuração é feita em três menus do Artisan, na ordem abaixo.

Config ▸ Device

Na aba ET/BT, selecione MODBUS como aparelho — é isso que faz o Artisan usar as leituras do Torrador.

Dependendo da sua versão do Artisan, o próprio menu Config ▸ Device só aparece com o modo Expert ligado em Config ▸ Mode ▸ Expert.

Mantenha todas as outras configurações com o valor padrão e pressione OK.

Config ▸ Port, aba MODBUS

Esta tela reúne os sensores de temperatura (em cima) e a conexão (embaixo).

Dependendo da versão, esta janela abre sozinha ao confirmar o MODBUS no passo anterior.

Sensores — na tabela Inputs (em cima), o Artisan lê o Input 1 como BT e o Input 2 como ET:

  • Input 1 → BT (grão): Device 1, Register 0, Function 4, Divider 10, Mode C, Decode uInt16.
  • Input 2 → ET (ar): Device 1, Register 1, Function 4, Divider 10, Mode C, Decode uInt16.

O Divider 10 converte o valor enviado (temperatura ×10) para graus. Para conferir, aqueça a ponteira do BT (grão) e veja se é o número BT que sobe no Artisan; se subir o ET, inverta o Register (0 e 1) entre os dois inputs.

Conexão (parte de baixo):

  • No campo Type (canto inferior direito), escolha TCP.
  • Host: o mesmo endereço que você usa para abrir esta página (ou torrador.local).
  • Port: a porta definida acima (padrão 502).

Mantenha todas as outras configurações com o valor padrão e pressione OK.

Config ▸ Events, aba Sliders

Aqui você configura o comando de potência do queimador, na linha Burner:

  • Marque a caixa no início da linha (coluna Event) para ativar o controle.
  • Action: Modbus Command.
  • Min 0, Max 100, Factor 1.0, Offset 0.00 — enviam a potência de 0 a 100 sem alterar.

Importante: o Artisan só se conecta com o Torrador salvo no modo de controle Artisan — é quando a porta MODBUS abre e o equipamento passa a obedecer aos comandos. Em Manual ou Automático a porta fica fechada e a conexão é recusada.

Limite máximo absoluto: vale em todos os modos e desliga o queimador (trava — exige BOOT). Deixe em branco para desativar.

Acesso ao site

Deixe a senha em branco para manter o site aberto — use apenas em redes confiáveis. Esqueceu a senha? Ela pode ser redefinida pela porta serial (comando auth).

A segurança das funções oferecidas pela página web depende diretamente da segurança da rede em que o torrador opera.

Saiba mais

Na prática, ela é tão protegida quanto essa rede: qualquer pessoa com acesso à rede pode acionar o equipamento, assim como qualquer pessoa próxima pode usar os botões físicos.

A senha acima é uma camada extra contra o acesso casual de outros aparelhos da rede — não um cofre. Para segurança rigorosa, evite conectar o torrador a uma rede aberta ou compartilhada: prefira o modo “Rede do Torrador” (AP), defina uma senha forte e mantenha conectados a essa rede privada apenas os dispositivos que devem operar o sistema.
